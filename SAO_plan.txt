Tâches restantes - MMO SAO (mod + map + serveur)
==============================================

Résumé global du projet
- Objectif : serveur Minecraft Forge 1.20.1 inspiré de Sword Art Online, jouable à grande échelle (~500 joueurs).
- Contenus attendus : hub Aincrad (ville départ), zones de farm, arènes de boss, HUD SAO complet, systèmes de progression (stats, XP custom), armes/armures/cristaux SAO, boss multi-phases, ressources custom (textures/sons).
- Contraintes : performance (optimisation serveur), scripts de gestion (build/deploy, backup/reset, start), compatibilité client/serveur Forge.

Mod (saoworld)
- Boss Illfang : ajouter phases (rage <50% HP), dash/cleave, adds, loot étoffé.
- HUD SAO avancé : party HP, buffs, icône crystal, log d’événements (kill, loot).
- Systèmes : stats joueur (force/vitesse), progression simple (XP custom), drops/craft SAO.
- Ressources : textures/sons custom (armes, cristaux, HUD), son d’interface SAO.
- Mobs : mobs SAO (loups, sangsues) + boss étage 2 (patterns, adds).

Map/Monde
- Étendre hub Aincrad : ville départ, structures, portails TP zones farm/boss.
- Zones farm : spots pour mobs SAO, loot équilibré.
- Arène boss : salle dédiée Illfang, future salle boss étage 2.
- Importer/placer une map Aincrad plus grande (open source) si disponible.

Serveur
- Scripts de gestion : backup/reset du monde, relance rapide.
- Start script simple (Java 17) déjà présent; optionnel: planifié au boot.

Prochaines actions (proposées)
- Priorité 1 : IA boss Illfang (phases rage <50% HP, dash/cleave, adds) + loot équilibré. (Partiel fait : enrage, dash, cleave, adds.)
- Priorité 2 : HUD SAO avancé (party/buffs/notifications) + textures/sons custom.
- Priorité 3 : Étendre hub/map (ville + zones farm + arène) ou importer une Aincrad open source.
- Progression : stats joueur (force/vitesse), XP custom, drops/craft SAO.
- Scripts : one-click build+copie+backup+start, start serveur dédié, backup/reset monde.
